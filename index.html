VSYNK

We sync multiple screens

VSYNK is a video sync application built with openframeworks 0.74 with IOS 10.7 SDK. Currently, VSYNK runs only on Mac OS X. There will be Windows version in the future releases. You can download and install VSYNK software package for free. Free version provides video sync up to two machines.
